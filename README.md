This commit includes the project source code, dataset preprocessing pipeline, exploratory data analysis (EDA), TF-IDF feature engineering, machine learning model training, evaluation, and dashboard visualisations.

Three classifiers were benchmarked—Logistic Regression, Multinomial Naive Bayes, and Linear Support Vector Machine (SVM). The Linear SVM achieved the best performance with an F1-score of 98.83%, misclassifying only 28 out of 2,400 held-out test articles.

The project demonstrates an end-to-end machine learning workflow, from data cleaning and feature engineering to model comparison, performance evaluation, and communicating insights through visualisations.
